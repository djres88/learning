1. INTRO: TEXT LEVELS

That header was made using an HTML element -- specifically, <h1> (the "opening tag") and </#h1> (the "closing tag"). Most HTML elements have an opening tag and a closing tag, with the difference being the presence of a "/" in the closing tag. In addition to h1 headers, there are other "levels" of header elements: h2, h3, up to h6 headers. There are paragraphs, too: p elements are the preferred method for normal-sized paragraph text on websites. You can create a p element like so:

<p>I'm an p tag!<p>

You can also escape the code make comments in html by bracketing text between the thingies <!-- and -->. Note: You can escape the code in other ways, too; for example, when you want to insert special characters AS TEXT (e.g. <) rather than have them trigger code. See http://text-symbols.com/html/ for more info on how to write code as text.

3. CSS

When you entered <h2 style="color: red">CatPhotoApp</h2>, you were giving that individual h2 element an inline style. Often the better way is by using CSS ("Cascading Style Sheets"). Besides color, there are hundreds of CSS properties you can use to change the way an element looks on your page.

To use CSS, first create a style element: <style></style>.

Inside that style element, you can create a CSS selector for all h2 elements. For example, if you wanted all h2 elements to be red, your style element would look like this:

<style>
  h2 {
   color: red;
  }
</style>.

Note the opening and closing curly braces ({ and }) around each element's style. Also be sure to add the semicolon to the end of each of your element's styles.
