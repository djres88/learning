1. INTRO: TEXT LEVELS

That header was made using an HTML element -- specifically, <h1> (the "opening tag") and </h1> (the "closing tag"). Most HTML elements have an opening tag and a closing tag, with the difference being the presence of a "/" in the closing tag. In addition to h1 headers, there are other "levels" of header elements: h2, h3, up to h6 headers. There are paragraphs, too: p elements are the preferred method for normal-sized paragraph text on websites. You can create a p element like so:

<p>I'm a p tag!<p>

You can also escape the code in html by bracketing text between <!-- and -->. This allows you to make comments on the code, without affecting the code itself.

NOTE: You can escape the code in other ways, too, which comes in handy for projects such as this website. In html, the characters < and & (among others) refer to code. If you want to enter these and other characters as TEXT, you need to do some special work. See http://text-symbols.com/html/ for more info on how to write code as text.

3. CSS

When you entered <h2 style="color: red">CatPhotoApp</h2> , you were giving that individual h2 element an inline style. Often the better way is to use CSS ("Cascading Style Sheets"). Besides color, there are hundreds of CSS properties you can use to change the way an element looks on your page.

To use CSS, first create a style element: <style></style>.

Inside that style element, you can create a CSS selector for all h2 elements. For example, if you wanted all h2 elements to be red, you would write something like this in the style section:

 <style>
   h2 {
    color: red;
   }
 </style>.

Note the opening and closing curly braces ({ and }) around each element's style. Also be sure to add the semicolon to the end of each of your element's styles.



4. STYLE CLASSES

Classes are reusable styles that can be added to HTML elements. Here's an example CSS class declaration:

 <style>
  .blue-text {
   color: blue;
  }

We've created a CSS class called blue-text within the <style> tag. Note the presence of a "." preceding the class name. Once you declare a class, you can apply it to an html element. For example, here we apply .blue-text to the h2 element:

<h2 class="blue-text">CatPhotoApp</h2>.

WHY IS THIS USEFUL? Instead of applying a single style to every h2 element — or dictating the styles for each individual header — you can "save" a class (a style setting) and call it as needed. Think of these classes as style templates.

Note that you can apply multiple classes to an element like so:

<p class = "class1 class2"> text goes here </p>. (all within THE SAME quotes)

5. FONT-SIZE & FONT-TYPE

You can also change the FONT SIZE using font-size and px (for example, h1 { font-size: 30px; }). You can change the FONT TYPE by using the font-family property. For example, if you wanted to set a style class "classy" with font size 16px and font type Sans-serif:

<style>
 .classy {
  font-size: 28px;
  font-family: 'Sigmar One', cursive;
  background-color: yellow;
 }

Now we could apply the class to an h2 header:

<h2 class="classy">CatPhotoApp</h2>

Becomes:

CatPhotoApp



6. IMPORTING FONTS

We can make calls to Google to load fonts into our HTML; the font we just used, Sigmar One, is an example. To add a font, copy the code snippet from Google and paste it into your code editor, above the styles (see the source code to this doc for an example). If we want to add, say, the Lobser font, we can write:

<link href="http://fonts.googleapis.com/css?family=Lobster" rel="stylesheet" type="text/css">.

Now you can set Lobster as a font-family value on your h2 (or any other) html element:

h2 {
 font-family: Lobster;
}

You can also specify how fonts should "degrade" if a font isn't available. Think of these as the "backup plan" fonts. For example, if you wanted an element to use the Helvetica font, but also degrade to the Sans-Serif font when Helvetica wasn't available, you could use this CSS style:

{ font-family: Helvetica, Sans-Serif; }.

For a complete list of fonts, please see the links at the bottom of the page (go to the link for Google Fonts).

8. Borders

CSS borders have properties like style, color and width. For example, if we wanted to create a red, 5 pixel border around an HTML element, we could define the class within the CSS style section: .thin-red-border {
 border-color: red;
 border-width: 5px;
 border-style: solid;
}

(a) Border Radius

Use to round corners. Do it with pixel value (border-radius: 10px;) OR percentage (border-radius: 50%;).
